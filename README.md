# 101fetch

## 101fetch displays 

* The logo
* The current time
* Your uptime


![101fetch in action](https://github.com/salaaad2/101fetch/blob/master/101fetch.png)

# Installation
* clone this repository anywhere you want :^)
* add the *following lines* to `.zshrc`
```
clear
/path/to/101fetch
```
* you're good to go
